# Svetly Angel — Travel Landing Page

## Концепция
Мобильный лендинг-путеводитель для путешествия троих друзей на Мадейру (апрель 2026).
Название: "Светлый Ангел"
Стиль: светлый, минималистичный, смешной, дружелюбный, современный.
Flat-иконки, кликабельные ссылки, геймификация (история путешествия как квест/приключение).
Защита: JS-пароль на входе. Хостинг: GitHub Pages.
Mobile-first, но работает и на десктопе.

## Стек
Один HTML-файл + встроенный CSS/JS. Leaflet.js для карты. Никаких фреймворков.
Бесплатные фото с Unsplash/Pexels.

## Секции лендинга

### 1. Экран входа
- Пароль (JS), фон с Мадейрой
- noindex/nofollow

### 2. Hero / Обратный отсчёт
- Название "Светлый Ангел"

- Виджет погоды на Мадейре

- Leaflet.js, маркеры каждой точки
- Линия маршрута
- Клик = описание + ссылка

### 4. Таймлайн по дням (главная секция)
Карточки-дни, скроллятся как сторис.

### 5. Бюджет
- Таблица расходов
- Итого на 1 / на 3

### 6. Чек-лист "Что взять"
- Интерактивные галочки (localStorage)
- Категории: палатка/кемпинг, одежда, документы, техника

### 7. Документы
- PDF кнопки: контракт машины, разрешения на кемпинг
- Ссылки на бронирования

### 8. Полезное

## Чек-лист "Что взять"
### Кемпинг
- [ ] Палатка (2 шт на 27-28)
- [ ] Спальник (на 10-15°C)
- [ ] Коврик/каремат
- [ ] Горелка + газ
- [ ] Котелок / турка
- [ ] Кофе, чай
- [ ] Нож / мультитул
- [ ] Фонарик / налобный фонарь
- [ ] Зажигалка / спички
- [ ] Мусорные пакеты (обязательно!)

### Одежда
- [ ] Дождевик / ветровка (обязательно!)
- [ ] Тёплая кофта / флис (ночью в горах 8-12°C)
- [ ] Треккинговые ботинки
- [ ] Шлёпки / сандалии (для бассейнов)
- [ ] Купальник
- [ ] Солнцезащитные очки
- [ ] Кепка / шляпа

### Документы
- [ ] Паспорт
- [ ] Водительские права (международные?)
- [ ] Страховка
- [ ] Распечатки бронирований
- [ ] Разрешения на кемпинг (PDF)

### Техника
- [ ] Powerbank
- [ ] Зарядка телефона
- [ ] Камера / GoPro
- [ ] Переходник розетки (Португалия = тип F, как в Европе)

### Полезное
- [ ] Солнцезащитный крем
- [ ] Бутылка для воды (1.5л минимум на человека для кемпинга)
- [ ] Аптечка (пластыри, обезболивающее)
- [ ] Полотенце быстросохнущее
