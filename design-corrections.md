# Svetly Angel — история правок Павла
> Записываем каждую правку. Потом анализируем и формируем чеклист для Design Engineer.

## 2026-03-25

### Типографика
- 12px для body текста — слишком мелко. Минимум 14px
- Заголовки секций увеличить до 28px
- Заголовки карточек 18px
- Основной текст 16px, дополнительный 14px
- Ссылки, адреса, время — тоже 14px, не мельче
- Цены в бейджах — 13px допустимо
- text-transform: uppercase на датах — убрать, выглядит крикливо

### Выравнивание
- Время в timeline — фиксированная ширина колонки, все описания по одной вертикали
- Текстовое время (День, Ночь, Утро) смещает — заменять на иконки одинаковой ширины
- Длинные интервалы ("06:30–13:45") смещают — сокращать до одного времени ("06:30")
- Адрес-ссылка — переносить на новую строку, не inline с названием

### Информационная архитектура
- Информация (факты, температура) и рекомендации (рестораны, советы) — РАЗНЫЕ блоки
- Информация = серый блок. Рекомендации = жёлтый блок
- Детали (телефон, бронь, PIN) — в строке события, не в отдельной секции "Детали"
- Контакты (телефон + сайт) — под событием одной строкой, accent-цветом
- Не дублировать: если информация уже в строке, не повторять в "Деталях" снизу

### Кнопки и иконки
- Кнопки без текста, только SVG иконки (навигация, бронирование, телефон)
- 40×40px, radius 12px, полупрозрачный фон
- Иконки внутри кнопок — 18px
- Убрать эмодзи из кнопок — только flat SVG
- Подложка badge (эмодзи дней) — без бордера
- Толщина иконок (stroke-width) одинаковая везде (1.5)
- Между кнопками отступ 20px

### Навигация
- Sticky навигация поверх карты (z-index 1000+)
- SVG иконки + текст в пунктах навигации
- Active state при скролле (scroll-spy)
- Без эмодзи в навигации

### Аккордеон
- Radius 24px
- Плавная анимация (max-height, не display:none)
- Стрелка — SVG chevron, плавный rotate
- Hover — лёгкий голубой оттенок
- Badge слегка увеличивается при открытии

### Карта
- Координаты проверять — точки не должны быть в океане
- Прямые линии polyline — добавлять промежуточные точки
- Стилизованные тайлы (CartoDB Positron), не стандартный OSM
- z-index карты ниже навигации

### Цвета
- Цены — зелёные бейджи
- Время — серый (text-secondary), НЕ accent
- Градиент hero — зелёно-голубой допустим, но НЕ оранжевый
- Hover — лёгкий голубой, не бежевый

### Контент
- "Автобус Минск → Вильнюс" — убрать "Автобус", и так понятно из контекста
- "Прибытие 06:30" — убрать если следующая строка начинается с 06:30
- Ecolines — сделать ссылкой
- Адреса — кликабельные (Google Maps)
- Телефоны — кликабельные (tel:)
- Табы с вариантами маршрутов — добавлять детальное описание каждого варианта
- Заезд/выезд — только время заезда + количество ночей, без лишних деталей
- Вильнюс → иконка волка 🐺, не замок

### Порядок кнопок
- Навигация (карта) — ВСЕГДА крайняя правая
- Другие кнопки (билет, бронирование) — левее
- Телефон — НЕ кнопка, а строка контактов под текстом

### Ссылки на сервисы
- Ecolines, Booking, кемпинг сайт — переносить на новую строку в contacts
- Не inline с названием

### Консистентность карточек
- Каждое место с названием → описание в em, кнопка навигации
- Каждое жильё → адрес-ссылка, заезд, кол-во ночей, контакты (тел + сайт), кнопки (бронь + навигация)
- Каждый ресторан/бар → описание, кнопка навигации
- Варианты маршрутов → табы с детальным описанием
- "Глава X:" — мелкий серый текст, название — bold основным

### Выравнивание строк
- Строки без описания: всё по центру (align-items: center)
- Строки с описанием/контактами: время по верху (flex-start), кнопки по центру (align-self: center)
- :has(em) и :has(.event-contacts) для определения строк с описанием

### Badge подложки
- Цвет подложки зависит от контекста иконки (badgeColor per day)
- Автобус/машина/хайкинг → жёлтый, Самолёт/океан → голубой, Волк/горы/город → серый, Туман/прогулка → зелёный
- Подложка НЕ меняется при открытии аккордеона

### Время
- font-weight 400 (как описание, не жирнее)
- "до 11:00" — "до" маленькой буквой, 12px через <small>
- День 1 · дата — тоже font-weight 400

### Шрифт
- DM Sans не поддерживает кириллицу → разные размеры латиницы и кириллицы
- Использовать шрифт с нативной кириллицей: Golos Text, Manrope, Commissioner
- Golos Text — лучший выбор (Паратайп, идеальная кириллица)

### Антипаттерны (запрещено)
- CSS clouds, animated gradient, glass-morphism
- Rainbow badges
- Шрифт Inter
- Эмодзи в навигации
- display:none для аккордеона
- 12px для body текста
